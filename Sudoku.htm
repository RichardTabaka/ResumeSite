Sudoku Solver Web App

Currently, this puzzle takes a Sudoku in the form of 81 characters, 0-9, converts them to an array of arrays in JavaScript, solves the puzzle, and displays the solution in an alert.

I would like to create a table of inputs so it's a little more intuitive for the user. Then, it would display the solution in a table on the page itself instead of in an alert. For now, though, this was mind-melting enough.
